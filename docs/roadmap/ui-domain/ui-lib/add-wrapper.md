# UI-Lib: add-wrapper

## Command

```
scaffold component add-wrapper <name> [--package ui-lib]
```

## Description

Add container component (hook + presentational).

## Injection Target

- **Artifact:** `src/components/<Name>/<Name>Container.tsx`

## Technology Stack

- **Container/presentational** pattern: wrapper = hook + presentational child. Effect in hook for data/effects.

Planned
