# UI-Lib: add-story

## Command

```
scaffold component add-story <name> [--package ui-lib]
```

## Description

Add Storybook story for a component.

## Injection Target

- **Artifact:** `src/components/<Name>/<Name>.stories.tsx`

## Co-Generation

- Depends on component existing

## Technology Stack

- **Storybook** for component docs; Tailwind/Shadcn styling visible in canvas. Alternatives: Ladle, Chromatic.

## Status

Planned
