# UI: init-shadcn

## Command

```
scaffold ui init-shadcn [--package ui]
```

## Description

Initialize Shadcn in package. Run Shadcn init, create `components.json`.

## Injection Target

- **Artifact:** `components.json`

## Technology Stack

- **Shadcn** init creates components.json; Tailwind preset. Alternatives: Radix manual setup, Chakra provider.

Planned
