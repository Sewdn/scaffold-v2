# UI: add-base-color

## Command

```
scaffold ui add-base-color <color> [--package ui]
```

## Description

Change base color (slate, zinc, etc.). Update `components.json`.

## Injection Target

- **Artifact:** `components.json`

## Technology Stack

- **Shadcn** components.json base color; Tailwind palette. Alternatives: Chakra colorMode, Radix themes.

## Status

Planned
