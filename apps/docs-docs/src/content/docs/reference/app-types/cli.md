## Overview

Command-line interface (Effect + Commander).

**Directory prefix:** `cli`

**Package:** `@workspace/app-cli`

## Naming example

`scaffold app <name> --type cli` → `apps/cli-<name>`

## Expansion commands

| Command | Description |
|---------|-------------|
| `scaffold cli add-command [options]` | Add a new command to a scaffolded CLI application |
| `scaffold cli add-service [options]` | Add a new service to the service layer of a scaffolded CLI application |
